# Project 015: Find the Smallest Number in a List

## Description
This project asks the user to enter multiple numbers and finds the smallest number in the list.  
It teaches beginners about lists, loops, and comparison.

## Skills Learned
- Lists  
- Loops  
- Conditional statements (`if`)  
- User input  

## How to Run
1. Open `smallest_number.ipynb` in Jupyter Notebook.
2. Run all cells.
3. Enter numbers separated by commas.
